Real vs Fake Images Dataset

This dataset contains a balanced collection of real and AI-generated images. It is ideal for training deep learning models to distinguish between authentic and fabricated visuals.

  • 10,000+ labeled images
  • Sources: Internet, GAN-generated fakes
  • Categories: Humans, Animals, Objects
  • Resolution: 1024x1024 pixels
  • Format: JPEG / PNG

This dataset enables fine-tuning of models like CNNs, EfficientNet, and Transformer-based architectures for high-accuracy classification tasks.

Morphed Faces Dataset

This specialized dataset focuses on detecting face morphing attacks, commonly used for biometric fraud. It provides a comprehensive set of morphed and original face images.

  • 5,000+ morphed and genuine face images
  • Techniques: Landmark-based morphing, Blending methods
  • Metadata included (morphing details)
  • Format: High-quality PNG images
  • Ground-truth masks for morphed regions

Perfect for developing and testing CNNs, autoencoders, or patch-based approaches for morphed face detection tasks.
